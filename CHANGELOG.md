# Журнал изменений

Формат основан на Keep a Changelog. Пока нет первого релиза, изменения собираются в разделе «Не выпущено».

## Не выпущено

### Добавлено

- Базовый проект Next.js 16 с TypeScript и зависимостями Prisma/PostgreSQL.
- Landing, auth skeleton и локальная проверка CSV/XLSX до 10 МБ без серверной отправки.
- Prisma schema и начальная миграция для пользователей, отчётов, строк, снимков прибыльности и тарифов.
- Финансовое ядро в integer-копейках, контракт парсера, определение WB/Ozon и синтетические fixtures.
- 21 unit-тест финансовых формул, определения маркетплейса и проверки файла.
- Проектная документация, план `Sprint 0: Foundation` и журнал архитектурных решений.
- CI для установки зависимостей, Prisma, форматирования, типизации, lint, тестов и сборки.
- Автоматические GitHub Issues для недельного планирования и вечерних отчётов.
- Шаблоны pull request и задачи Codex.
- Оригинальный маскот Profit Doctor с подсказками по ходу landing.

### Изменено

- Формула прибыли собрана в читаемое уравнение «выручка − расходы = прибыль по SKU» с расшифровкой расходов.
- Увеличены кегль и контраст текста в списке проблем, карточках аудитории и тарифных возможностях.
- Тарифы теперь явно наследуют возможности предыдущего уровня; у каждого пункта есть доступная информационная подсказка.
- Демо-диагностика на первом экране получила интерфейс карточки продукта и комментарий Profit Doctor.

### Исправлено

- Текст во всех трёх карточках аудитории начинается на одном уровне.
- Карточка Start снова получает выделенный фон и сохраняет общую геометрию тарифной сетки.

### Безопасность

- Зафиксированы правила работы с отчётами продавцов, секретами и трассировкой финансовых расчётов.
- `npm audit` проходит без известных уязвимостей; транзитивный PostCSS закреплён на исправленной версии.
