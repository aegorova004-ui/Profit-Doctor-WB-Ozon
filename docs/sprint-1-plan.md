# Sprint 1: Первый полезный импорт

План обновлён 13 июля 2026 года. Статус: **в работе; первый локальный WB preview-импорт реализован**.

## Цель

Дать пользователю проверяемый результат по одному формату Wildberries, не создавая ложной уверенности в точности прибыли и не отправляя чувствительный отчёт на сервер без необходимости.

## Срез 1 — WB XLSX preview

- [x] Подключить чтение XLSX в браузере без выполнения формул и макросов.
- [x] Читать десятичные суммы строками и переводить их в integer-копейки через `BigInt`.
- [x] Распознавать API-подобный набор колонок WB и версию `wb:api-financial-report:preview-2026-07`.
- [x] Обрабатывать продажи, возвраты, комиссию, логистику, хранение, штрафы, доплаты и удержания.
- [x] Агрегировать операции по SKU и сверять расчёт с `for_pay` до копейки.
- [x] Показывать локальную сводку и таблицу SKU с пометкой «оценка до себестоимости и рекламы».
- [x] Останавливать неизвестный или несошедшийся формат без вывода полной строки отчёта.
- [x] Покрыть binary fixture, граничные значения денег и React upload-сценарий автоматическими тестами.

## Срез 2 — подтверждённый формат и себестоимость

- [ ] Получить актуальную полностью анонимизированную выгрузку из кабинета WB; текущий fixture синтетический и не закрывает этот пункт.
- [ ] Сравнить заголовки, знаки и контрольные суммы с preview-контрактом; создать отдельную production-версию адаптера.
- [ ] Добавить ввод или импорт себестоимости по SKU с явным источником значения.
- [ ] Показать расшифровку каждой суммы и статус неполных данных.
- [ ] Добавить флаг убыточного SKU после учёта себестоимости.

## Срез 3 — результат и экспорт

- [ ] Добавить demo report и удобный повторный запуск.
- [ ] Реализовать CSV/XLSX-экспорт результата с версией формулы и источниками полей.
- [ ] Проверить большой файл, мобильную таблицу и основной сценарий smoke-тестом.
- [ ] Решить, нужна ли серверная история отчётов; до отдельного решения файлы остаются только в браузере.

## Контрольный расчёт fixture

Синтетический файл содержит 4 операции и 3 SKU. Ожидаемые итоги:

- выручка — 6 840 ₽;
- известные удержания и возвраты — 3 062,50 ₽;
- остаток до себестоимости и рекламы — 3 777,50 ₽.

Итог равен сумме `for_pay` по всем операциям. Это техническая сверка, а не показатель реального магазина.

## Критерий готовности Sprint 1

Актуальная анонимизированная выгрузка WB проходит подтверждённый адаптер без ручной правки; суммы сходятся с ручным расчётом; себестоимость и источники показаны явно; неполный результат называется оценкой; результат можно экспортировать.
