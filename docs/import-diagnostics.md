# Диагностика импорта

Обновлено: 21 июля 2026 года.

Документ задаёт карту поддерживаемых форматов и публичные сообщения ошибок для импорта. Он нужен, чтобы парсеры, UI и тесты говорили одним языком и не раскрывали чувствительные строки отчёта.

| WB          | `wb:finance-report:api-csv:preview-2026-07` | preview | `wb-finance-large-preview.csv`           | 36 операций, 36 SKU, длинная таблица и мобильные карточки без блокировки прокрутки               |

## Категории ошибок

| Категория                      | Типичные коды                                                                                             | Что произошло                                                | Что сказать пользователю                                                                                         | Что предложить дальше                                     |
| ------------------------------ | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------- |
| Неподдерживаемый формат        | `FORMAT_NOT_RECOGNIZED`, `WB_FINANCE_CSV_FORMAT_NOT_RECOGNIZED`, `OZON_FINANCE_CSV_FORMAT_NOT_RECOGNIZED` | Не нашли обязательные признаки поддерживаемого отчёта        | «Не нашли финансовый отчёт WB/Ozon. Проверьте, что файл выгружен из раздела финансовых отчётов»                  | Скачать демо-шаблон или выбрать другой CSV/XLSX           |
| Товарный каталог вместо отчёта | `WB_PRODUCT_CATALOG_UPLOADED`                                                                             | В файле есть товарные поля, но нет выплат и удержаний        | «Похоже, это товарный каталог Wildberries. Для расчёта прибыли нужен финансовый отчёт с выплатами и удержаниями» | Скачать WB XLSX/CSV демо или выгрузить финансовый отчёт   |
| Не хватает колонок             | `MISSING_COLUMNS`                                                                                         | Файл похож на нужный отчёт, но обязательных колонок нет      | «В отчёте не хватает колонок для расчёта: …»                                                                     | Проверить шаблон, период выгрузки и настройки экспорта    |
| Повторяются колонки            | `DUPLICATE_COLUMN`                                                                                        | Один заголовок встречается несколько раз                     | «В отчёте повторяется колонка „…“. Сохраните файл заново или удалите дубль»                                      | Сохранить исходный отчёт заново                           |
| Неверная валюта                | `UNSUPPORTED_CURRENCY`                                                                                    | Валюта не `RUB`/`RUR`                                        | «Сейчас поддерживаем только рублёвые отчёты»                                                                     | Выгрузить отчёт в рублях                                  |
| Неверная сумма                 | `INVALID_MONEY`                                                                                           | Денежное поле нельзя безопасно перевести в копейки           | «В строке № … сумма в колонке „…“ не похожа на число»                                                            | Проверить формат числа: `1250`, `1250,50`                 |
| Неверное количество            | `INVALID_QUANTITY`                                                                                        | Количество не целое                                          | «В строке № … количество должно быть целым числом»                                                               | Проверить строку и выгрузить файл заново                  |
| Неверный знак операции         | `INVALID_OPERATION_SIGN`                                                                                  | Продажа/возврат не совпадает со знаком количества или суммы  | «В строке № … знаки суммы и количества не совпадают с типом операции»                                            | Проверить исходный отчёт; если это сторно — нужен адаптер |
| Неподдерживаемая операция      | `UNSUPPORTED_OPERATION`, `UNSUPPORTED_OZON_OPERATION`                                                     | Парсер не знает, как безопасно разнести операцию по SKU      | «В строке № … есть операция, которую preview-адаптер пока не распределяет по товарам»                            | Отправить анонимизированный пример операции               |
| Только сервисные строки        | `NO_PRODUCT_FINANCE_ROWS`                                                                                 | В файле нет товарных строк с SKU                             | «В файле нет товарных строк с SKU. Общие услуги нельзя честно распределить по товарам без правил»                | Выгрузить отчёт с продажами товаров                       |
| Сервисные строки без SKU       | warning `SERVICE_ROWS_WITHOUT_SKU_SKIPPED`                                                                | Есть общие услуги без товара                                 | «В отчёте есть сервисные строки без SKU. Мы не распределяли их по товарам»                                       | Сверить итог вручную, если сумма существенная             |
| Конфликт идентификаторов SKU   | `SKU_ID_CONFLICT`                                                                                         | Один SKU связан с разными идентификаторами marketplace       | «Для SKU „…“ найдены разные идентификаторы. Расчёт остановлен, чтобы не смешать товары»                          | Проверить карточки и выгрузку                             |
| Слишком большой файл           | `TOO_MANY_ROWS`, `TOO_MANY_COLUMNS`, upload validation                                                    | Файл превышает текущий лимит строк, колонок или размера      | «Файл слишком большой для preview-импорта»                                                                       | Выгрузить меньший период                                  |
| XLSX не читается               | `XLSX_READ_FAILED`, `CSV_READ_FAILED`                                                                     | Файл повреждён, в неверной кодировке или не является отчётом | «Не удалось прочитать файл. Сохраните отчёт заново в XLSX или CSV UTF-8»                                         | Сохранить заново и повторить                              |

## UX-правила для ошибок

- Заголовок ошибки должен отличать проблему: не использовать универсальное «Ошибка».
- Текст ошибки строится так: что нашли → почему расчёт остановлен → что сделать дальше.
- Номер строки можно показывать, если он есть. Полную строку, ФИО, названия кабинетов, телефоны, email и другие чувствительные поля показывать нельзя.
- Если ошибка связана с форматом, рядом нужна ссылка на демо-шаблоны.
- Если ошибка связана с preview-ограничением, нужно прямо писать «preview-адаптер», чтобы не обещать production-поддержку.

## Следующие UI-состояния

1. Неизвестный файл: карточка «Формат пока не поддерживается» + список найденных признаков + ссылки на демо WB/Ozon.
2. Не хватает колонок: карточка «Не хватает колонок для расчёта» + список недостающих колонок + совет выгрузить файл заново.
3. Неподдерживаемая операция: карточка «Операция требует проверки» + номер строки + просьба дать анонимизированный пример.
4. Service-only файл: карточка «Нет товарных строк» + объяснение, почему общие услуги не распределяются на SKU автоматически.
